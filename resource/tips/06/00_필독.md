# Module 06 필독 사항

## 의도적인 예외 발생

이번 모듈에는 Subject PDF에서 의도적으로 예외를 발생시키도록 설계된 과제가 포함되어 있습니다.

대표적으로 다음 과제가 있습니다.

- Alembic 4
- Kaboom 1

위 과제들은 정상 동작 여부가 아니라

"특정 상황에서 예외가 발생하는 이유"

를 이해하는 것이 목적입니다.

따라서 traceback이 출력되더라도 반드시 코드가 잘못된 것은 아닙니다.

## mypy 관련

Alembic 4에서는 의도적으로 존재하지 않는 속성에 접근합니다.

예시:

```python
alchemy.create_earth()
```

이 경우 mypy는 attr-defined 오류를 발생시킬 수 있습니다.

필요 시 다음과 같이 무시할 수 있습니다.

```python
alchemy.create_earth()  # type: ignore[attr-defined]
```

## helper 실행 테스트

helper는 Subject PDF의 예시 출력 기준으로 테스트를 수행합니다.

일부 과제 실행테스트는 예외 발생 자체를 정상 결과 처리 합니다.
