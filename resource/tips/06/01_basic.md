# Module 06 개요

## Python Import System

이번 모듈에서는 Python 코드 구조화의 핵심인 Import System을 학습합니다.

작은 프로젝트에서는 모든 코드를 한 파일에 작성할 수 있습니다.

하지만 프로젝트가 커질수록 코드를 여러 파일로 분리해야 합니다.

Python은 이를 위해

- Module
- Package
- Import

기능을 제공합니다.

## __init__.py 파일

이번 모듈의 핵심은 __init__.py 입니다.

단순히 패키지를 생성하는 역할뿐 아니라

- 함수 공개(export)
- 별칭(alias)
- 패키지 인터페이스 구성

등을 담당하게 됩니다.

결과적으로 모듈06번은 Python Import System을 학습하기 위한 모듈입니다.

- import
- package
- __init__.py
- relative import
- circular import

개념을 이해하는 것이 중요합니다.

이번 모듈의 목표는

"코드를 여러 파일로 분리하고 서로 연결하는 방법 이해"

입니다.
