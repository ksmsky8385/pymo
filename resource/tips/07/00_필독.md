# Module 07 필독 사항

## 최소 파일 구조

이번 모듈은 Subject PDF에 파일 구조가 명확하게 지정되어 있지는 않습니다.

따라서 helper는 실행용 루트 파일과 각 ex 폴더의 `__init__.py`만 생성합니다.

```text
python_module_07/
├── battle.py
├── capacitor.py
├── tournament.py
│
├── ex0/
│   └── __init__.py
│
├── ex1/
│   └── __init__.py
│
└── ex2/
    └── __init__.py
```

클래스와 기능을 나눌 내부 파일은 자신의 설계에 맞춰 자유롭게 추가할 수 있습니다.

## helper 자동 생성 구조

helper는 아래 상수 기준으로 파일을 생성합니다.

```bash
MODULE_07_FILES=(
  "battle.py"
  "capacitor.py"
  "tournament.py"
  "ex0/__init__.py"
  "ex1/__init__.py"
  "ex2/__init__.py"
)
```

## ex 폴더 내부 파일

ex 폴더 내부에 추가하는 파일명은 자유롭게 정할 수 있으며 `MODULE_07_FILES`를 수정할 필요가 없습니다.

helper의 파일 생성 상태와 빈 파일 표시는 루트 실행 파일 3개와 각 ex 폴더의 `__init__.py`만 기준으로 판정합니다.

실행 테스트에서는 루트의 `battle.py`, `capacitor.py`, `tournament.py`를 순서대로 실행합니다.
