# Module 07 필독 사항

## 권장 파일 구조

이번 모듈은 Subject PDF에 파일 구조가 명확하게 지정되어 있지는 않습니다.

따라서 아래 구조는 추천 구조이며, 다른 방식으로 구현해도 무방합니다.

```text
python_module_07/
├── battle.py
├── capacitor.py
├── tournament.py
│
├── ex0/
│   ├── __init__.py
│   ├── base.py
│   ├── creatures.py
│   └── factories.py
│
├── ex1/
│   ├── __init__.py
│   ├── base.py
│   ├── creatures.py
│   └── factories.py
│
└── ex2/
    ├── __init__.py
    ├── base.py
    ├── strategies.py
    └── exceptions.py
```

## helper 자동 생성 구조

helper는 아래 상수 기준으로 파일을 생성합니다.

```bash
MODULE_07_FILES=(
  "battle.py"
  "capacitor.py"
  "tournament.py"
  "ex0/__init__.py"
  "ex0/base.py"
  "ex0/creatures.py"
  "ex0/factories.py"
  "ex1/__init__.py"
  "ex1/base.py"
  "ex1/creatures.py"
  "ex1/factories.py"
  "ex2/__init__.py"
  "ex2/base.py"
  "ex2/strategies.py"
  "ex2/exceptions.py"
)
```

## 파일명을 변경하는 경우

파일명은 자유롭게 변경 가능합니다.

예를 들어

```text
ex1/base.py
```

대신

```text
ex1/capabilities.py
```

를 사용해도 됩니다.

하지만 helper는 위 상수값을 기준으로 파일을 생성하므로

반드시

```bash
MODULE_07_FILES
```

세팅용 상수의 배열도 함께 수정해야 합니다.

그렇지 않으면 helper 파일 생성 기능에서 -Created 표시 및

실행테스트에서 파일 인식 애러가 날 수도 있습니다.
