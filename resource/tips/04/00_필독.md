# Module 04 필독 사항

## 실행 테스트 및 리소스 안내

본 헬퍼는 Subject PDF에 포함된 예시 실행 환경을 최대한 동일하게 재현하여 자동 테스트를 수행합니다.

실행 테스트에 필요한 파일들은 헬퍼 내부에 포함되어 있으며, 자동으로 복사되어 사용됩니다.

사용되는 주요 테스트 파일은 다음과 같습니다.

- ancient_fragment.txt
- etc/master.passwd

`ancient_fragment.txt`는 파일 읽기 및 저장 기능 검증에 사용됩니다.

Subject PDF의 예시에서는 `/etc/master.passwd` 절대경로를 사용하는 경우가 있습니다.

42경산의 클러스터 PC에는 해당 경로에 해당 파일이 존재하지 않으므로 helper 실행테스트에서는 임의의 `etc/master.passwd` 파일을 ex폴더에 생성, `master.passwd`파일 접근 권한을 제거 후 Permission Error 처리를 검증합니다.

만약 코드 내부에서 `/etc/master.passwd` 경로를 사용하는 경우 `etc/master.passwd`로 변경하셔야 실행 테스트가 원할히 작동합니다.

특정 과제의 경우 실행 후 `new_fragment.txt`파일이 생성됩니다.

helper의 실행 테스트 종료 후 테스트 산출물을 감지하는 로직이 실행됩니다.

산출물 삭제를 원치 않을 시 `n`를 입력하여 파일을 보존하셔야 합니다.
