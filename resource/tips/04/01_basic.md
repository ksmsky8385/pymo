# Module 04 개요

## Python File Handling & Stream Management

이번 모듈에서는 파일(File)을 읽고 쓰는 방법과 파일 시스템에서 발생하는 다양한 예외 상황을 처리하는 방법을 학습합니다.

실제 프로그램은 항상 정상적인 입력만 받지 않습니다.

- 존재하지 않는 파일
- 접근 권한이 없는 파일
- 잘못된 경로
- 저장 실패
- 읽기 실패

등 다양한 문제가 발생할 수 있습니다.

이번 모듈에서는 이러한 상황을 안전하게 처리하면서 파일을 다루는 방법을 익히게 됩니다.

## 이번 모듈의 핵심 도메인 기술

### File Open

파일을 열고 읽기 또는 저장을 수행하는 기능입니다.

### File Object

파일을 열면 반환되는 객체입니다.

파일의 데이터를 읽고 저장하는 작업을 담당합니다.

### Exception Handling

파일 처리 과정에서 발생하는 예외를 처리합니다.

프로그램이 비정상 종료되지 않도록 보호합니다.

### File System Errors

파일이 존재하지 않거나 접근 권한이 없는 경우 발생하는 예외입니다.

대표적으로 다음 예외를 만나게 됩니다.

- FileNotFoundError
- PermissionError

### Standard Streams

운영체제가 제공하는 기본 입출력 채널입니다.

- stdin
- stdout
- stderr

### Context Manager

파일 자원을 안전하게 관리하기 위한 Python 기능입니다.

### with Statement

파일을 자동으로 정리(close)하기 위한 Python 권장 문법입니다.

## 목표

파일은 운영체제가 관리하는 중요한 자원(Resource)입니다.

잘못 관리하면 데이터 손실이나 자원 누수가 발생할 수 있습니다.

이번 모듈에서는

- 파일 읽기
- 파일 저장
- 예외 처리
- 표준 스트림
- Context Manager

를 활용하여 안전하고 안정적인 파일 처리 프로그램을 작성하는 방법을 학습합니다.

이번 모듈의 목표는

**"파일을 안전하게 다루고 예외 상황에서도 안정적으로 동작하는 프로그램 작성"** 입니다.
