# 클래스 기초 및 인스턴스

파이썬에서 클래스는 객체(Object)를 만들기 위한 **설계도(틀)** 이며, 이를 통해 생성된 객체를 **인스턴스(Instance)** 라고 합니다.

## 1. 기본 구조와 생성자 (`__init__`)

``` python
class 클래스이름:
    def __init__(self, 인자1: 타입, 인자2: 타입) -> None:
        self.속성1 = 인자1  # 인스턴스 변수 정의
        self.속성2 = 인자2
```

- __init__ 메서드: 인스턴스가 생성될 때 자동으로 호출되는 초기화 함수입니다.

- `self`의 원리: 현재 생성되고 있는 '객체 자신'을 가리킵니다. 클래스 내에서 속성이나 메서드에 접근할 때는 항상 self.을 붙여야 합니다.

## 2. 인스턴스 메서드 (Instance Method)

``` Python
class 클래스이름:
    def 메서드이름(self) -> 리턴타입:
      ...
```
원리: 클래스 내부의 함수는 첫 번째 인자로 무조건 self를 받아야 합니다.

호출할 때: 객체에서 메서드를 호출할 때는 self 인자를 생략합니다. (파이썬이 내부적으로 현재 객체를 self에 자동으로 전달함)

``` Python
# 정의는 def show(self): 로 했지만
cactus.show()  # 호출 시에는 인자를 비워둠
```

`self`가 붙은 변수(`self.name`)는 객체가 살아있는 동안 유지되는 **'객체의 기억'**이고, 일반 변수(`name = self.name`)는 해당 메서드가 끝나면 사라지는 **'임시 변수'**입니다.
