# Module 01 개요

## 객체지향 프로그래밍(Object-Oriented Programming)

현실 세계의 사물은 이름, 상태, 행동을 가집니다.

파이썬의 객체지향 프로그래밍(OOP)은 이러한 현실의 개념을 코드로 표현하기 위한 방법입니다.

이번 모듈에서는 정원 관리 시스템을 예제로 사용하여 식물(Plant)을 객체로 모델링하고 객체 간의 관계를 설계하는 방법을 학습합니다.

## 이번 모듈의 핵심 도메인 기술

* **클래스와 인스턴스 (`class`)**
  - 객체를 생성하기 위한 설계도 작성
  - 인스턴스 생성 및 속성 관리

* **생성자 (`__init__`)**
  - 객체 생성 시 초기 상태 설정
  - 인스턴스 변수 초기화

* **인스턴스 메서드**
  - 객체가 수행할 행동 정의
  - `self`를 통한 객체 상태 관리

* **캡슐화 (Encapsulation)**
  - Getter / Setter를 이용한 데이터 보호
  - 잘못된 값이 저장되지 않도록 검증

* **상속 (Inheritance)**
  - 부모 클래스의 기능 재사용
  - 공통 기능을 공유하며 코드 중복 제거

* **메서드 오버라이딩 & `super()`**
  - 부모 기능 확장 및 재정의
  - 자식 클래스만의 동작 구현

* **클래스 메서드와 스태틱 메서드**
  - 클래스 단위 기능 제공
  - 객체와 독립적인 유틸리티 구현

이번 모듈의 목표는 "객체지향 프로그래밍의 핵심 원리를 이해하고, 클래스와 상속을 활용하여 유지보수하기 쉬운 프로그램을 작성하는 것" 입니다.
