# Module 03 개요

## Python Collections & Data Processing

이번 모듈에서는 Python이 제공하는 다양한 컬렉션(Collection) 자료형과 데이터 처리 기법을 학습합니다.

같은 데이터라도 어떤 자료구조를 선택하느냐에 따라 코드의 가독성, 성능, 활용 방법이 크게 달라집니다.

이번 모듈에서는 게임 데이터 분석 환경을 예제로 사용하여 데이터를 효율적으로 저장하고 처리하는 방법을 익힙니다.

## 이번 모듈의 핵심 도메인 기술

### Command Line Arguments (`sys.argv`)

프로그램 실행 시 외부에서 전달된 데이터를 읽어옵니다.

CLI 환경에서 데이터를 입력받고 처리하는 기본 방법을 학습합니다.

### List

가장 기본적인 컬렉션 자료형입니다.

- 순서 보장
- 인덱스 접근 가능
- 수정 가능

여러 데이터를 저장하고 순회하며 통계를 계산하는 데 사용됩니다.

### Tuple

고정된 데이터를 표현하기 위한 자료형입니다.

- 순서 보장
- 수정 불가 (Immutable)

좌표 정보와 같이 변경되면 안 되는 데이터를 표현할 때 사용됩니다.

### Set

중복을 허용하지 않는 집합 자료형입니다.

- 중복 제거
- 합집합 (Union)
- 교집합 (Intersection)
- 차집합 (Difference)

데이터 비교 및 집합 연산을 수행할 때 매우 유용합니다.

### Dictionary

Key-Value 구조로 데이터를 저장합니다.

```python
inventory = {
    "potion": 5,
    "sword": 1,
}
```

빠른 데이터 조회와 관리가 가능하며,
실무에서도 가장 많이 사용되는 자료구조 중 하나입니다.

### Generator (`yield`)

필요한 순간에만 데이터를 생성하는 특별한 함수입니다.

- 메모리 사용량 절약
- 대량 데이터 처리
- 스트림 처리

모든 데이터를 한 번에 저장하지 않고
필요할 때마다 생성하여 사용할 수 있습니다.

### Comprehension

컬렉션을 간결하게 생성하는 Python 문법입니다.

- List Comprehension
- Dictionary Comprehension
- Set Comprehension

반복문과 조건문을 한 줄로 표현하여
데이터 변환 및 필터링을 효율적으로 수행할 수 있습니다.

## 자료구조 선택 기준

| 자료형 | 특징 |
|--------|--------|
| List | 순서가 있으며 수정 가능 |
| Tuple | 순서가 있으며 수정 불가 |
| Set | 중복 제거 및 집합 연산 |
| Dictionary | Key-Value 데이터 관리 |
| Generator | 필요 시점에 데이터 생성 |

데이터의 특성에 맞는 자료구조를 선택하고,

컬렉션 자료형과 Generator, Comprehension을 활용하여

효율적이고 읽기 쉬운 데이터 처리 코드를 작성하는 능력을 기릅니다.
